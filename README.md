# big-clock

A fullscreen web clock for Windows with:

- large centered clock display
- 12/24 hour toggle
- adjustable size
- floating quick timers and custom timer
- alarm sound + visual shake on timer completion
- multiple clock styles including default, LED neon, 7-segment, flip clock, and retro tech

## Run

Open `index.html` in a browser.

## Controls

- `F` — fullscreen
- `T` — toggle 12/24 hour
- `Q` — toggle quick timers panel
- `W` — toggle custom timer panel
- `Esc` — stop active alarm
